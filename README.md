Aim:
To perform data binning and data formatting using Python.

Theory:
Data binning, also known as discretization or bucketing, involves converting continuous numerical data into a set of discrete categories or intervals called bins. Instead of handling a wide range of raw numerical values, binning groups them into a smaller number of meaningful categories, making the data easier to interpret and analyze.

The Pandas library offers a variety of functions to support binning, text formatting, data type conversion, rounding, sorting, and category analysis. These operations are commonly applied in areas like business analytics, sales evaluation, and operational data handling to simplify data representation and enhance decision-making.

Key functions include:

DataFrame(): Constructs a structured dataset in tabular form from input data such as dictionaries.
cut(): Divides continuous numerical data into discrete bins or categories.
astype(): Converts the data type of a specific column.
dtypes: Displays the data types of all columns in the dataset.
str.upper(): Converts all text values in a column to uppercase.
str.lower(): Converts all text values in a column to lowercase.
round(): Rounds numeric values to a specified number of decimal places.
sort_values(): Arranges data based on one or more selected columns.
unique(): Identifies and returns distinct values in a column.
value_counts(): Calculates the frequency of each unique value.
print(): Outputs the processed data or results.

Conclusion:
The implementation demonstrates that data binning and formatting using Python and the Pandas library can effectively organize, standardize, and simplify raw data, making it more suitable for analysis and informed decision-making.
